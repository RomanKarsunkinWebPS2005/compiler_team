# Список тестов

## Класс Lexer

### 🔹 Базовая структура
- [x] Ключевое слово начала программы: `bello!`
- [x] Игнорирование пробелов и переводов строк
- [x] Разделение токенов пробелами и переносами строк

---

### 🔹 Ключевые слова
- [x] Все ключевые слова из спецификации
- [x] Разделение ключевых слов от идентификаторов
- [x] Чувствительность к регистру

---

### 🔹 Идентификаторы
- [x] Простые имена, подчёркивание, цифры после первой позиции
- [ ] Token с типом Error
---



### 🔹 Типы данных (зарезервированы)
- [x] `Banana`, `Papaya`, `Gelato`, `Spaghetti`

---

### 🔹 Литералы
- [x] Числовые: целые, отрицательные, дробные
- [x] Строковые: `!...!`, экранирование `!!`, последовательность `\\n`

---

### 🔹 Операторы
- [x] Оператор присваивания: `x lumai 5`
- [x] Оператор сложения: `a melomo b`
- [x] Оператор вычитания: `a flavuk b`
- [x] Оператор умножения: `a dibotada b`
- [x] Оператор деления: `a poopaye b`
- [x] Оператор остатка от деления: `a pado b`
- [x] Оператор возведения в степень: `a beedo (b Banana c)` 
- [x] Оператор равенства: `x con y`
- [x] Оператор меньше: `x la y`
- [x] Оператор больше: `x looka too y`
- [x] Оператор логического не: `makoroni flag`
- [x] Оператор логического и: `a tropa b`
- [x] Оператор логического или: `a bo-ca b`
- [x] Проверка приоритета операций(умножение над сложением): `a melomo b dibotada c`

---

### 🔹 Прочие лексемы
- [x] Скобки: `(`, `)`
- [x] Завершение выражений: `naidu!`
- [x] Комментарии: `//`, `/* */`

---
